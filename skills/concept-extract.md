# 结构化笔记 · 概念提取

## 任务

针对**已经整理好的结构化内容**（markdown 笔记、文章、逻辑梳理稿、文档等），你的工作是：

1. **保留原文**：原始内容一字不改地原样输出
2. **追加词典**：在原文末尾追加一个「专业名词解释词典」段落，提取所有学术/专业概念

## 核心规则

- **不改写原文**，原样输出
- **不要做总结、改写、压缩**
- 只在末尾**新增**一个 `## 专业名词解释词典` 段落
- 至少提取 5-20 个概念（如果原文涉及的概念少，最少也要提取 3 个）

## 提取标准

**必须提取**（可穿越周期的概念）：
- **学术/理论**：贝叶斯推理、涌现、第一性原理、证伪、存在主义、信息论、熵
- **数学/物理**：马尔可夫链、博弈论、热力学第二定律、混沌理论、临界现象
- **经济/心理/哲学**：马太效应、锚定效应、禀赋效应、沉没成本、机会成本、现象学
- **AI/技术**：Transformer、RLHF、Scaling Law、MoE、强化学习、对齐
- **社会/组织理论**：科层制、博弈均衡、制度经济学、网络效应
- **从原文抽离出的理论框架**：如果作者自己提出了一个可迁移的模型（如"社会债务 = 价值 - 价格"），也算

**不提取**：
- 公司名、产品名（字节、阿里、ChatGPT）
- 具体人物（除经典理论提出者）
- 临时的描述性词语
- 作者仅在本文使用一次的非通用术语
- 过于笼统的词（学习、成功、思考）

## 输出格式（严格遵守）

```
[原文完整内容，一字不改]

---

## 专业名词解释词典

**[术语1]**（English Name）定义或解释内容。要求至少一句话的完整定义，说明这个概念是什么、为什么重要。

**[术语2]**（English Name 2）定义...

**[术语3]**（没有英文名的话省略括号）定义...

...（直到提取完所有概念）
```

**严格要求**：
- 必须以 `## 专业名词解释词典` 作为段落标题（用两个 `#`，前后有空行）
- 每个术语必须用 `**[术语名]**` 格式包围（带中括号和双星号）
- 有英文的一定要带英文（用中文括号 `（English）`）
- 每个术语必须有完整的定义，**不能只列名字**
- 术语之间用空行分隔
- 最后不要再添加任何其他内容（如总结、尾注）
