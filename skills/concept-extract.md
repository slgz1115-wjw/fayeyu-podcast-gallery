# TL;DR + 概念词典提取（两段输出，不重复原文）

## 任务

对下面这份已经整理好的结构化内容，**输出两个段落**，顺序固定：

1. `## TL;DR` —— 核心观点 + 阐述逻辑 + 叙述脉络（300-500 字）
2. `## 专业名词解释词典` —— 可穿越周期的专业术语

**不要重复原文**、不要做总结、不要添加其他任何章节。系统会自动把原文拼接在这两段中间。

## TL;DR 写作要求

放在整个输出最前面，严格按三段结构：

**核心观点**：用 1-3 句话点破文章最根本的论断、判断或结论。不是"本文讨论了 X"，而是"关于 X，核心论断是 Y"。

**阐述逻辑**：用 4-8 句话梳理文章的论证链条，用"因为 A，所以 B；进一步地因为 B，所以 C"的因果结构。不要罗列章节，而是抽象出推理骨架：前提 → 中间论点 → 结论。

**叙述脉络**：用 1-2 句话描述文章是以什么结构展开的（如：时间纵轴→当下横切；问题→分析→建议；案例铺垫→理论提炼；多角度论证→反常识收尾）。

**硬性要求**：
- 总长度 300-500 字
- 是逻辑的抽象，不是摘要的复述
- 不用"本文讨论了"这种套话，直接给判断和结构

---

## 专业名词提取标准

**必须提取**（可穿越周期的概念）：
- 学术/理论：贝叶斯推理、涌现、第一性原理、证伪、存在主义、信息论、熵
- 数学/物理：马尔可夫链、博弈论、热力学第二定律、混沌理论、临界现象
- 计算机科学：冯诺依曼架构、时间复杂度、哈希表、TCP/IP、微服务、CI/CD、事件循环
- 经济/心理/哲学：马太效应、锚定效应、禀赋效应、沉没成本、机会成本、现象学
- AI/技术：Transformer、RLHF、Scaling Law、MoE、强化学习、对齐
- 社会/组织理论：科层制、博弈均衡、制度经济学、网络效应
- 从原文抽离出的可迁移理论框架（如「社会债务 = 价值 − 价格」）

**不提取**：
- 公司名、产品名（字节、阿里、ChatGPT）
- 具体人物（除经典理论提出者）
- 仅在本文使用一次的描述性词语
- 过于笼统的词（学习、成功、思考）

**数量要求**：最少 8 个，最多 30 个。如果原文很长、涉及很多领域，一定要覆盖全部主要概念。

## 输出格式（严格遵守）

你的整个回答必须严格为以下格式，**不要有前言、不要有收尾、不要重复原文**：

```
## TL;DR

**核心观点**：...

**阐述逻辑**：...

**叙述脉络**：...

---

**[术语2]**（English Name 2）定义...

**[术语3]**（没有英文名则省略括号）定义...
```

## 严格要求

- 必须以 `## TL;DR` 开头，中间 `---` 分隔，然后 `## 专业名词解释词典`
- 两个 `##` 标题必须一字不差，它们是下游系统解析锚点
- 每个术语必须用 `**[术语名]**` 格式包围（中括号+双星号）
- 有英文的用中文括号 `（English）`
- 每个术语必须有完整的一句话以上的定义，**不能只列名字**
