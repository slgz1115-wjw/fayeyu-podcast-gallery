## 正文

姐妹们/兄弟们，分享一个我自己做的播客学习工具，真的太好用了 🎙

### 痛点
每次听完一期很好的播客（晚点聊、硬地骇客、什么是快乐星球...），里面有很多让我醍醐灌顶的观点，但过几天就忘了 😭 笔记又懒得手动做，就算做了也是散装的...

### 我做了什么
用 Claude Code 搭了一个「播客知识工作站」，核心能力就三个字：**全！自！动！**

### 工作流程
1️⃣ 输入播客名字 → 自动搜索并订阅 RSS
2️⃣ 新一期更新 → 系统自动通知
3️⃣ 点一个按钮「自动提炼」→ 后面全部自动：
   - 下载音频
   - Groq Whisper 极速转录（1小时播客2分钟转完！）
   - Claude AI 自动提炼三模块笔记

### 笔记长什么样？
不是那种敷衍的 AI 总结！是真正有深度的结构化提炼：
- **反共识金句**：10-20 条「听起来反常识但仔细一想确实有道理」的观点
- **分章节核心观点**：完整还原嘉宾的论述逻辑，不是干巴巴的列表
- **专业名词词典**：所有听不懂的术语都有解释

### 最 amazing 的是知识图谱！！！
所有播客里提到的专业概念，自动按 6 大领域分类：
🔬 底层原理（贝叶斯、奥卡姆剃刀...）
🧠 AI 理论（Transformer、强化学习...）
⚙️ AI 工程（CUDA、MCP...）
🤖 AI 应用（Agent、具身智能...）
🚀 商业战略（ARR、PMF...）
📈 经济金融（马太效应、期权...）

而且是**双向链接**的！点一个概念就能跳到笔记里它出现的位置，像 Obsidian 一样 ✨

### 技术栈
- 前端：纯 JS（轻量快速）
- 转录：Groq Whisper API（免费极速）
- 提炼：Claude Code CLI
- 知识图谱：自动分类 + 可编辑
